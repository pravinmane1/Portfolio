About

I'm a software engineer who believes great front-end work is invisible — when it's done right, nobody notices. They just get things done faster.

I've been building web applications professionally since 2020, starting at SLB where I cut my teeth on enterprise Angular and .NET Core. There, I led the component restructuring of a flagship application, improving load performance through dynamic data fetching and NgRx state management.

Since 2024, I've been at JP Morgan Chase & Co., building and maintaining Angular applications at banking scale — where correctness, accessibility, and performance aren't optional. I work across the full TypeScript / Angular / .NET stack, with AWS infrastructure.

Before all this, I was the kid in 2014 teaching myself HTML and CSS on a borrowed laptop, just to see something I built appear on a screen. That same curiosity still drives me today.
